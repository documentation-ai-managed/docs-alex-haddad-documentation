## Overview

Crivvi lets you securely share sensitive information like passwords, API keys, and files using zero-knowledge encryption. You maintain full control without exposing data to third parties. Follow these steps to create an account, invite your team, and share your first secret in under five minutes.

## Create Your Account

Sign up takes seconds. Use your email or social login.

<Steps>
  <Step title="Visit Crivvi" icon="globe">
    Go to [app.crivvi.com](https://app.crivvi.com) and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="edit-3">
    Provide your email and a strong password. Verify with the email link sent to `your.email@company.com`.
  </Step>
  <Step title="Complete Setup" icon="check-circle">
    Choose a username like `alex-haddad` and enable two-factor authentication (2FA).
  </Step>
</Steps>

<Callout kind="tip">
  Enable 2FA immediately for added security. Crivvi supports TOTP apps like Authy or Google Authenticator.
</Callout>

## Invite Your Team

Build your team by inviting members with role-based access.

<Tabs>
  <Tab title="Web Dashboard" icon="monitor">
    <Steps>
      <Step title="Navigate to Teams">
        Click **Teams** in the sidebar.
      </Step>
      <Step title="Add Member">
        Enter `team member@company.com` and select **Viewer** or **Editor** role.
      </Step>
      <Step title="Send Invite">
        They receive a secure link expiring in 24 hours.
      </Step>
    </Steps>
  </Tab>
  <Tab title="API" icon="code">
    Use the invitations endpoint.

    <CodeGroup tabs="cURL,JavaScript">
      ```bash
      curl -X POST https://api.crivvi.com/v1/teams/invite \
        -H "Authorization: Bearer YOUR_API_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "email": "team member@company.com",
          "role": "editor"
        }'
      ```

      ```javascript
      const response = await fetch('https://api.crivvi.com/v1/teams/invite', {
        method: 'POST',
        headers: {
          'Authorization': 'Bearer YOUR_API_TOKEN',
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({
          email: 'team member@company.com',
          role: 'editor',
        }),
      });
      const data = await response.json();
      console.log(data);
      ```
    </CodeGroup>
  </Tab>
</Tabs>

## Share Your First Secret or File

Send encrypted data that only recipients can decrypt.

<Steps>
  <Step title="Create Share" icon="plus">
    Click **New Share** and choose **Secret** or **File**.
  </Step>
  <Step title="Add Content" icon="edit">
    For secrets, enter `DB_PASSWORD: supersecret123!`. For files, upload `config.json`.
  </Step>
  <Step title="Set Options" icon="settings">
    Set expiration to `7 days`, max views to `5`, and add recipients.
  </Step>
  <Step title="Share Link" icon="share-2">
    Copy the secure link: `https://crivvi.com/s/abc123xyz` and send via email or Slack.
  </Step>
</Steps>

<Response tabs="Success">
```json
{
  "id": "share_abc123xyz",
  "url": "https://crivvi.com/s/abc123xyz",
  "expires_at": "2024-11-01T12:00:00Z",
  "max_views": 5
}
```
</Response>

## Initial Configuration

Customize your workspace for better security and usability.

<ExpandableGroup>
  <Expandable title="API Tokens" default-open="true">
    Generate tokens for integrations.

    <ParamField path="api_key" param-type="string" required="true">
      Your primary authentication token. Keep it secret.
    </ParamField>

    <CodeGroup tabs="Bash,Python">
      ```bash
      export CRIVVI_API_KEY="sk_live_abc123def456ghi789"
      curl https://api.crivvi.com/v1/shares -H "Authorization: Bearer $CRIVVI_API_KEY"
      ```

      ```python
      import os
      os.environ['CRIVVI_API_KEY'] = 'sk_live_abc123def456ghi789'
      import requests
      response = requests.get('https://api.crivvi.com/v1/shares', headers={
        'Authorization': f'Bearer {os.environ["CRIVVI_API_KEY"]}'
      })
      ```
    </CodeGroup>
